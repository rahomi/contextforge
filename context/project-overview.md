# ContextForge - Project Overview

## Product Purpose
ContextForge is an AI-powered developer productivity platform that helps engineers understand, document, and improve software projects through context engineering and intelligent agents.

The platform combines:
1. **Developer Productivity Dashboard** - Repository insights, code health metrics, and engineering activity tracking
2. **Personal Engineering Knowledge Base** - Documentation, technical decisions, and learning resources
3. **AI Engineering Assistant** - Intelligent analysis, documentation generation, and development support

## Target Users
- **Software Engineers** - Individual contributors who want to understand codebases better
- **Technical Leads** - Team leads who need visibility into project health and architecture
- **Engineering Managers** - Managers who want to track team productivity and technical debt
- **Open-Source Maintainers** - Maintainers who need to document and manage community projects
- **AI Agent Users** - Developers who use AI coding assistants and need better project context

## Main Workflows
1. **Repository Analysis** - Connect repositories, analyze structure, and track health
2. **Knowledge Management** - Create, organize, and retrieve engineering documentation
3. **AI-Powered Insights** - Generate documentation, understand code, and get recommendations
4. **Engineering Activity Tracking** - Monitor commits, PRs, and development progress
5. **Context Sharing** - Provide AI agents with reliable project knowledge

## Core Features
### Module 1: Developer Productivity Dashboard
- Repository insights (metadata, languages, frameworks, dependencies)
- Code health tracking (coverage, quality, technical debt, security)
- Engineering activity display (PRs, commits, milestones)

### Module 2: Personal Engineering Knowledge Base
- Markdown-based documentation storage
- Search and tagging system
- Category organization
- Version history
- Related document linking

### AI Assistant Features
- Repository understanding and analysis
- Documentation generation
- Engineering support and recommendations
- Code review assistance

## Non-Goals
- **Not a CI/CD platform** - Focus on insights and knowledge, not build pipelines
- **Not a project management tool** - No task tracking or sprint management
- **Not a code hosting platform** - Integrate with existing repos, don't replace GitHub/GitLab
- **Not a real-time collaboration tool** - Focus on individual and async workflows

## Future Possibilities
- Team collaboration features
- Integration with more AI models
- Browser extension for context injection
- VS Code extension
- Slack/Teams integration for notifications
- API for third-party integrations
- Advanced analytics and reporting
- Custom workflow automation
